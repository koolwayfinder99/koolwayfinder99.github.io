The Project: Warehouse AMR

The Problem

Existing open-source navigation stacks failed in highly dynamic warehouse environments, leading to path failures when forklifts or people crossed the robot's trajectory.

The Solution

I re-architected the navigation stack using ROS2 Humble. I implemented a custom behavior tree for dynamic obstacle avoidance and tuned the TEB Local Planner. This resulted in a robust system capable of replanning paths in < 200ms.

Tech: ROS2, Nav2, Behavior Trees, LiDAR

The Project: Surface Defect Detection

The Problem

Manual quality control on the steel production line was slow and had a 12% error rate.

The Solution

I built a Computer Vision pipeline using Python and OpenCV. I trained a lightweight CNN model using TensorFlow to classify surface scratches. The system was deployed on an edge device (Jetson Nano) and achieved 98% detection accuracy in real-time.

Tech: Python, TensorFlow, OpenCV, Edge Computing

The Project: High-Speed Manipulator

The Problem

The standard Python-based controller for our 6-DOF arm was inducing 50ms of latency, causing jitter in precision tasks.

The Solution

I rewrote the control loop in modern C++ (C++17) to manage memory manually and utilize real-time kernel capabilities. I implemented a custom Inverse Kinematics solver using the KDL library, reducing cycle time to 1ms.

Tech: C++, KDL, Real-time Systems

The Project: Drone Stabilization

The Problem

A quadcopter prototype was unstable during payload delivery due to shifting centers of gravity.

The Solution

I designed a cascade PID controller to handle the attitude and position loops separately. By fine-tuning the derivative gains and implementing an Extended Kalman Filter (EKF) for sensor fusion, I achieved stable flight under varying load conditions.

Tech: PID, EKF, MATLAB/Simulink
